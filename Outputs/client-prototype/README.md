# 好运彩店客户端原型

基于 `Imports/ReferenceImage2.jpg`、`ReferenceImage3.jpg` 和 `ReferenceImage4.jpg` 搭建的 390 x 844 移动端交互原型。

## 已实现

- 大厅：活动 Banner、账户余额、店主与邀请入口、合买/跟单切换、彩种入口
- 比赛：运动/玩法/日期切换、实时比分、赔率选择、投注单、金额调整、模拟提交
- 个人：实名状态、店铺、余额隐藏、充值/提现弹层、订单统计、服务入口
- 全局：三栏底部导航、操作反馈、移动端无横向溢出、减少动态效果支持

这是前端原型，不包含真实登录、支付、出票、开奖、持久化或后端 API。

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev
```

生产构建：

```powershell
npm.cmd run build
```
